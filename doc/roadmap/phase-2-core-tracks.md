# Phase 2 — Core Tracks (Weeks 5–20)

## Structure

Six tracks run in parallel. Each week targets specific modules across tracks. Each module produces a working project — not just notes.

**Module rhythm:** ~10–15 hours per module. 3–4 modules per month per track.
